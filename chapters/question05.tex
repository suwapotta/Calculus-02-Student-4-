\subsubsection*{Problem Statement Question 5}
The following triple integral represents the \textbf{force cost function} in a bounded musculoskeletal system with quadratic variation along the vertical axis:
\[ \iiint_{E}(480.92x+592.03y+z^{2})dV \]
where $E$ is the solid tetrahedron bounded by: $x=0.1$, $y=0.3$, $z=0.4$, $x+y+z=5$. Evaluate the \textbf{total force cost} within the system.

\subsubsection*{Solution}
To set up the triple integral, we first determine the bounds of integration for the solid tetrahedron $E$:
\begin{itemize}
	\item \textbf{Inner bounds ($z$):} Bounded below by $z=0.4$ and above by the plane $z=5-x-y$.
	\item \textbf{Middle bounds ($y$):} $ z_{\text{min}} = 0.4 \Rightarrow y \in{[ 0.3,\ 5 - 0.4 - x]} \iff y \in{[ 0.3,\ 4.6 - x]}  $
	\item \textbf{Outer bounds ($x$):} $ y_{\text{min}} = 0.3,\ z_{\text{min}} = 0.4 \Rightarrow x \in{[ 0.3,\ 5 - 0.3 - 0.4]} \iff x \in{[ 0.1,\ 4.3]} $
\end{itemize}

The total force cost $C$ is calculated by:
\[ C = \int_{0.1}^{4.3} \int_{0.3}^{4.6-x} \int_{0.4}^{5-x-y} (480.92x + 592.03y + z^2) \, dz \, dy \, dx \]

Applying the iterated integration process step-by-step:

\vspace{1cm}
\noindent\textbf{Step 1: Integrate with respect to $z$.}

\[ \int_{0.4}^{5-x-y} (480.92x + 592.03y + z^2) \, dz = \left[ (480.92x + 592.03y)z + \frac{z^3}{3} \right]_{0.4}^{5-x-y} \]
\[ = (480.92x + 592.03y)(4.6-x-y) + \frac{(5-x-y)^3}{3} - \frac{0.064}{3} \]
The integral expression becomes:
\[ C = \int_{0.1}^{4.3} \int_{0.3}^{4.6-x} \Bigg( (480.92x + 592.03y)(4.6-x-y) \]
\[ \qquad + \frac{(5-x-y)^3}{3} - \frac{0.064}{3} \Bigg) \, dy \, dx \]

\vspace{1cm}
\noindent\textbf{Step 2: Integrate with respect to $y$.}

Evaluating the expanded polynomial with respect to $y$ from $0.3$ to $4.6-x$ gives:
\[ \int_{0.3}^{4.6-x} \Big( ... \Big) \, dy = 240.46(x^3 - 9.2x^2 + 21.16x) + \frac{592.03}{6}(4.6-x)^3 \]
\[ \qquad + \frac{(4.7-x)^4}{12} + 144.276x^2 - 615.365x - 117.217 \]
The integral expression simplifies to:
\[ C = \int_{0.1}^{4.3} \Bigg( 240.46(x^3 - 9.2x^2 + 21.16x) + \frac{592.03}{6}(4.6-x)^3 \]
\[ \qquad + \frac{(4.7-x)^4}{12} + 144.276x^2 - 615.365x - 117.217 \Bigg) \, dx \]

\vspace{1cm}
\noindent\textbf{Step 3: Integrate with respect to $x$.}

\[ \int_{0.1}^{4.3} \Big( ... \Big) \, dx = \Bigg[ 240.46\left(\frac{x^4}{4} - \frac{9.2x^3}{3} + 10.58x^2\right) - \frac{592.03(4.6-x)^4}{24} - \frac{(4.7-x)^5}{60} \]
		\[ \qquad + 48.092x^3 - 307.6825x^2 - 117.217x \Bigg]_{0.1}^{4.3} \]

\vspace{1cm}
\noindent\textbf{Step 4: Simplify the result.}

Evaluating the upper and lower bounds:
\[ C \approx \boxed{16,732.3117} \]
