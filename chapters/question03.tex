\subsubsection*{Problem Statement Question 4}
In a multivariable musculoskeletal model, the \textbf{force output density function} for a limb system is:
\[ P(x, y, z) = 901.17xyz \]
where $x, y, z$ represent normalized inputs for muscle contraction, joint pressure, and bone displacement. A transformation to new coordinates is defined by:
\[ x=\frac{u+3v+w}{7}, \quad y=\frac{u-v+w}{7}, \quad z=\frac{u-v-3w}{7} \]
Compute the \textbf{total musculoskeletal force output} over the region:
$0 \le u \le 3$, $-1 \le v \le 6$, $-1 \le w \le 4$, using the \textbf{Jacobian transform}.

\subsubsection*{Solution}
The total musculoskeletal force output $F$ is calculated by evaluating the triple integral of the density function over the given region, translating to the $uvw$-space:
\[ F = \iiint_{G} P(x(u,v,w), y(u,v,w), z(u,v,w)) |J| \, du \, dv \, dw \]

First, we compute the Jacobian determinant $J$ of the transformation:
\[ J = \frac{\partial(x,y,z)}{\partial(u,v,w)} = \left(\frac{1}{7}\right)^3 \begin{vmatrix} 1 & 3 & 1 \\ 1 & -1 & 1 \\ 1 & -1 & -3 \end{vmatrix} \]
\[ = \frac{1}{343} \Big[ 1((-1)(-3) - (1)(-1)) \]
		\[ \qquad - 3((1)(-3) - (1)(1)) + 1((1)(-1) - (-1)(1)) \Big] \]
\[ = \frac{1}{343} \Big[ 1(4) - 3(-4) + 1(0) \Big] = \frac{16}{343} \approx 0.046647 \]

The absolute value of the Jacobian is $|J| = \frac{16}{343}$. Let $A = u - v + w$. Substituting the transformation into the integral yields:
\[ F = 0.122557 \int_{-1}^{4} \int_{-1}^{6} \int_{0}^{3} \left( A^3 + 4(v-w)A^2 - 16vwA \right) \, du \, dv \, dw \]

Applying the iterated integration process step-by-step:

\vspace{1cm}
\noindent\textbf{Step 1: Integrate with respect to $u$.}

\[ \int_{0}^{3} \left( A^3 + 4(v-w)A^2 - 16vwA \right) \, du = \left[ \frac{A^4}{4} + \frac{4(v-w)A^3}{3} - 8vwA^2 \right]_{u=0}^{u=3} \]
Expanding the bounds results in the following polynomial for $v$ and $w$:
\[ = 9v^3 + (21w - 22.5)v^2 - (21w^2 + 27w - 9)v - (9w^3 + 22.5w^2 + 9w - 20.25) \]
\[ \Rightarrow F = 0.122557 \int_{-1}^{4} \int_{-1}^{6} \Big( 9v^3 + (21w - 22.5)v^2 \]
\[ \qquad - (21w^2 + 27w - 9)v - (9w^3 + 22.5w^2 + 9w - 20.25) \Big) \, dv \, dw \]

\vspace{1cm}
\noindent\textbf{Step 2: Integrate with respect to $v$.}

Evaluating the polynomial with respect to $v$ from $-1$ to $6$:
\[ \int_{-1}^{6} \Big( ... \Big) \, dv = -63w^3 - 525w^2 + 983.5w + 1585.5 \]
The integral expression simplifies to:
\[ F = 0.122557 \int_{-1}^{4} \left( -63w^3 - 525w^2 + 983.5w + 1585.5 \right) \, dw \]

\vspace{1cm}
\noindent\textbf{Step 3: Integrate with respect to $w$.}

\[ \int_{-1}^{4} \left( -63w^3 - 525w^2 + 983.5w + 1585.5 \right) \, dw = \left[ \frac{-63w^4}{4} - 175w^3 + 491.75w^2 + 1585.5w \right]_{-1}^{4} \]
\[ = (-1022) - (-934.5) = -87.5 \]

\vspace{1cm}
\noindent\textbf{Step 4: Simplify the result.}

\[ F = 0.122557 (-87.5) \approx \boxed{-10.7237} \]
